# Ecoleta
## Conteudo
  * Ferramentas utilizadas:
      * Backend:
        - NodeJs com Express.
        - Knex com Postgres.
        
      * Frontend:
        - React router dom (navegação)
        - Leaflet (Mapa).
        - React Icons (icones/svg)
      
      * Mobile:
        - Expo.
        - Expo mail-composer (Envio de emails).
        - react-native-maps (Mapas).
        - Expo-location (GPS/Localização)
